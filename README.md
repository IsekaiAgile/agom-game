# アガイルド〜炎の王国〜

**AGOM (Agairud Game Of Management)**

過労で倒れたITエンジニアが、異世界「アガイルド」で「アジャイルな働き方」を学び直すファンタジーRPG。

---

## 🎮 ゲーム概要

### ストーリー
ITエンジニアの藤田雄真（ふじた ゆうま）は、過労で意識を失う。目覚めるとそこは謎の空間「転生の間」。試練を乗り越えた先で、竜のアジャドラが現れ、異世界「アガイルド」への転生を導かれる。

炎の王国アガイルドで、彼は本当の「働き方」と「生き方」を学んでいく――。

### テーマ
- 過労社会への警鐘
- アジャイルな働き方の学び
- 自分を大切にすることの重要性
- チームワークと成長

## 🎬 プレイ方法

### オンラインプレイ
https://isekaiagile.github.io/agom-game/

### ローカル実行
```bash
# インストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

- **Dキー**: デバッグメニューを開く

## 🎨 アセット

### BGM
- `dark-bgm.mp3`: プロローグ（暗い雰囲気）
- `fantasy-bgm.mp3`: タイトル、アジャドラシーン（ファンタジー）
- `trial-bgm.mp3`: 転生の間（神秘的）

### 背景画像
- 現実世界: 部屋、電車、オフィス
- 異世界: アガイルド、転生の間

---

## 🎮 操作方法

### キーボード
- **矢印キー（← →）**: 移動
- **スペースキー**: ジャンプ
- **クリック**: テキスト送り・選択肢・スタート
- **Dキー**: デバッグメニュー

### タッチ操作（モバイル）
- **画面左1/3**: 左移動
- **画面右1/3**: 右移動
- **画面中央上部**: ジャンプ
